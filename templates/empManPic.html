The Home Page

Description

The Home Page features two options: the Start/End Work and the Management Info/Admin Page.

Clock Page

Description

Clicking on the Start/End Work option redirects you to this page. On this page, you can Start or End your work shift and log your clock-in time.

Clock-in Code

Description

On clicking Start/End you will then need to enter your unique employee code, which would then Start or End your shift.

Management Info/Admin Page

Description

This section contains information about management and administrative tasks and is accessible only to admins or authorized personnel.

Verify Admin Page

Description

Clicking 'Management Info' on the Home Page will bring up this page, where you need to enter your admin ID (the employee code of the admin).
